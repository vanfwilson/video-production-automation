# Video Production Automation System

Fully automated video production pipeline using n8n, Claude AI, YouTube, and Canva.

## Features

- Workflow 1: Raw video → YouTube transcription → AI editing instructions
- - Workflow 2: Finished video → YouTube publishing with multilingual support
  - - Time saved: 2-3 hours per video
    - - Cost: ~$0.10 per video
